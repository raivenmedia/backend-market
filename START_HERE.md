## 🚀 Setup Instructions

### Prerequisites
- [ ] Node.js (v14+)
- [ ] npm
- [ ] MongoDB (Atlas or local)
- [ ] Stripe account (test mode)

### Step 1: Backend Setup
```bash
cd backend
npm install
cp .env.example .env
# Edit .env with MongoDB URI and Stripe keys
npm run dev
# Backend runs on http://localhost:5000
```

### Step 2: Frontend Setup
```bash
cd frontend
npm install
npm run dev
# Frontend runs on http://localhost:3000
```

### Step 3: Test
- Open http://localhost:3000
- Register and login
- Add products
- Test checkout

**See SETUP_CHECKLIST.md for detailed verification!**

## 💡 Tips

### For Development
- Keep both backend and frontend running
- Check browser console for errors (F12)
- Check terminal output for API errors
- Use provided sample data for testing

### For Customization
- UI colors: Edit tailwind.config.js
- API settings: Edit .env file
- Add features: Follow existing patterns
- Database: Easy to extend

### For Production
- Change NODE_ENV to production
- Use strong JWT_SECRET
- Setup MongoDB backup
- Enable HTTPS
- Configure firewall
